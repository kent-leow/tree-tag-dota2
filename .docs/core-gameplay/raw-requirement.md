🌲 TREE TAG CLASSIC — FULL GAME SPEC (ENT vs INFERNAL)
🎮 Core Fantasy

Two asymmetric teams:

🌳 Ents (Survivors) → survive, grow economy, build defenses, outscale
🔥 Infernals (Hunters) → hunt, burn forests, break bases, eliminate ents

Win conditions:

Ents: survive until timer ends OR defeat all infernals (variant dependent)
Infernals: eliminate all ents before timer ends

🔥 INFERNAL FACTION (HUNTERS)
Core identity
Strong early game
Map pressure + vision control
Anti-economy (destroy trees, disrupt hiding)
🧠 Infernal design pillars
Mobility + gap closing
AoE tree destruction
Detection / scouting
Snowball power (gets stronger as ents die)
👹 Typical Infernal units (classic archetypes)
1. Infernal Tank
High HP frontline
Melee cleave
Passive fire aura (burn trees nearby)
2. Infernal Stalker
High movement speed
Blink / charge ability
Single-target burst
3. Infernal Caster
AoE fire spells
Vision reveal / detection
Area denial
🔥 Core Infernal abilities (consistent across Tree Tag versions)
🌋 1. Immolation / Burning Aura
Burns nearby trees automatically
Constant map pressure
💨 2. Charge / Blink / Leap
Gap closing tool
Used to catch ents hiding in forest
🔥 3. Fire Wave / AoE burn
Clears forests quickly
Forces movement
👁️ 4. Reveal / Scouting ability
Temporary vision
Detect invisible / hidden ents (variant dependent)
💀 Infernal win loop
Find ent → destroy trees → isolate → burst → snowball → repeat
🌳 ENT FACTION (SURVIVORS)
Core identity
Weak early
Strong scaling economy
Terrain manipulation (trees = life)
Survival + macro gameplay
🧠 Ent design pillars
Hide in forests
Build economic engine
Create defensive structures
Scale into late-game guardians
🌱 Core ent mechanics
🌲 1. Tree dependency system
Forest = camouflage + resource + safety
Cutting trees is BOTH:
risk (exposure)
necessity (build space / resources)

Ents generate resources via:

Passive income (time-based)
Resource buildings
Tree regrowth mechanics
Harvesting / lumber conversion
🏗️ 3. Building system

Ents can construct:

🪵 Economy buildings
Resource storage (income multiplier)
Lumber depot / nature shrine
🛡️ Defensive buildings
Walls / root barriers
Healing grove
Slow fields (nature traps)
⚔️ Military buildings
Guardian spawners
Ent defenders / treants
Ancient towers
🌿 4. Regrowth mechanic
Trees can regrow over time or via ability
Critical for stealth + survival
🌳 Ent abilities (classic archetypes)
🌱 1. Root / Hide
Become invisible in forest
High stealth value
🌿 2. Tree regrowth
Recreates forest tiles
Anti-infernal counterplay
